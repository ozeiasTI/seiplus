🚀 Primeiros Passos

O SEI PLUS é um sistema avançado projetado para facilitar a criação, gerenciamento e visualização de fluxos processuais complexos. Com uma interface intuitiva e recursos poderosos como o autocomplete dinâmico e visualização em fluxograma, ele transforma a maneira como você organiza seus processos.


✨

📊

⚡
Autocomplete
Ganhe produtividade usando atalhos de teclado para preencher textos repetitivos instantaneamente.

📝 Como Criar um Fluxo

Siga estes passos simples para cadastrar um novo processo no sistema:

1

Informações Básicas

Preencha o Nome do Processo (ex: Licitação), o Subtipo (ex: Pregão Eletrônico) e uma Descrição opcional que ajude a identificar o objetivo do fluxo.

2

Adicionar Etapas

Clique no botão + Adicionar Etapa. Você pode adicionar quantas etapas forem necessárias para completar o seu processo.

3

Configurar as Etapas

Para cada etapa, defina o nome, o conteúdo e o tipo. Se for uma etapa de Decisão ou Triagem, não esqueça de configurar as opções de resposta e seus destinos.

4

Salvar o Fluxo

Após revisar todas as etapas, clique em Salvar Fluxo. O processo ficará disponível imediatamente na sua biblioteca.

Tipos de Etapas Disponíveis

Tipo Ícone Descrição Campos Principais
Texto 📝 Modelo de documento ou instrução textual. Título, Conteúdo, Obs
Anexo 📎 Indicação de documento que deve ser juntado. Nome do Anexo, Obs
Decisão 🔀 Pergunta de Sim/Não que ramifica o fluxo. Pergunta, Próximas Etapas
Triagem 🔍 Múltiplas opções de escolha para o usuário. Pergunta, Lista de Opções
Link 🔗 Acesso direto a recursos externos ou sites. URL, Descrição
Alerta ⚠️ Avisos importantes ou bloqueios no processo. Mensagem, Nível de Alerta
E-mail 📧 Modelo pronto para envio de comunicações. Assunto, Corpo do E-mail
Checklist ✅ Lista de verificação de itens obrigatórios. Itens da Lista
⚠️ Dica de Especialista: Ao usar Decisão ou Triagem, certifique-se de que todas as opções levam a uma etapa válida ou ao "Fim do Fluxo" para evitar erros no fluxograma.

⚡ Recurso de Autocomplete

O autocomplete é a ferramenta secreta para quem busca velocidade. Ele permite inserir parágrafos inteiros apenas digitando um atalho.

Como Ativar: Dentro de qualquer campo de texto, digite a barra invertida \ e comece a digitar o nome do seu atalho. Um menu suspenso aparecerá com as sugestões.

Gerenciando seus Atalhos

  1. Acesse o menu ⚙️ Configurações.
  2. Clique em Gerenciar Atalhos.
  3. Cadastre uma categoria (ex: "Cláusulas"), um nome curto (ex: "prazo") e o texto completo que deve ser inserido.
  4. Clique em + Adicionar Atalho para salvar.

👁️ Visualização e Exportação

O SEI PLUS oferece flexibilidade total na hora de consultar seus processos:


📋 Modo Lista

Ideal para o dia a dia. Mostra as instruções detalhadas, permite copiar textos com um clique e marcar itens do checklist.

📊 Modo Fluxograma

Gera um diagrama visual automático usando Mermaid.js. Perfeito para entender ramificações e decisões complexas.

📥 Exportação: Você pode baixar o fluxograma como imagem PNG clicando no botão de download na visualização do fluxo.

💾 Segurança dos Dados

Como o SEI PLUS funciona localmente, você é o dono dos seus dados. Gerencie-os na aba de Configurações:

Ação O que faz? Quando usar?
💾 Fazer Backup Gera um arquivo .json com todos os seus dados. Sempre! Faça backups regulares.
📂 Restaurar Carrega dados de um arquivo de backup. Ao trocar de computador ou recuperar dados.
📥 Mesclar Dados Adiciona novos fluxos sem apagar os atuais. Ao receber fluxos de colegas.
🔄 Resetar Tudo Apaga tudo e volta ao estado original. Cuidado! Use apenas se quiser limpar tudo.

🚀 Novidades da Versão 2.0

🔀 Fluxos Não Lineares

Agora você pode criar bifurcações, loops e múltiplos caminhos independentes. Cada etapa controla explicitamente para onde o fluxo deve seguir.

📄 Geração de PDF

Selecione múltiplos fluxos na tela de processos e gere um documento PDF consolidado com todas as instruções e modelos de texto.

🧠 Assistente de IA

Use o poder do ChatGPT para encontrar fluxos e montar documentos. Configure sua chave API na aba de Configurações.

❓ Perguntas Frequentes

O sistema precisa de internet para funcionar? ▼
Não! Após o primeiro acesso, o SEI PLUS funciona totalmente offline. Seus dados nunca saem do seu computador, garantindo total privacidade e segurança. A internet só é necessária para o Assistente de IA.
Como compartilho um fluxo específico com um colega? ▼
Vá na página 📚 Processos, marque o checkbox do fluxo que deseja compartilhar e clique em Exportar Selecionados. Envie o arquivo gerado para seu colega, e ele poderá importar via Configurações.
Posso editar um fluxo depois de salvo? ▼
Sim! Na lista de processos, clique no ícone do lápis ✏️ no card do fluxo. Isso carregará todo o processo no editor para que você possa fazer as alterações necessárias.
O que são as variáveis dinâmicas {nome}? ▼
São marcadores que você pode colocar nos textos. Quando você visualiza o fluxo, eles servem como lembretes de informações que precisam ser preenchidas manualmente naquele momento.

💡 Boas Práticas

✅ Faça

  • Use nomes claros e objetivos para as etapas.
  • Crie atalhos para textos longos de fundamentação legal.
  • Faça backup semanalmente em local seguro.
  • Use o Modo Fluxograma para validar a lógica do processo.

❌ Evite

  • Criar fluxos excessivamente longos (mais de 30 etapas).
  • Esquecer de definir a "Próxima Etapa" em decisões.
  • Limpar os dados do navegador sem ter um backup salvo.
