Política de Privacidade - SEIPLUS

A extensão SEIPLUS respeita a privacidade dos usuários e não coleta informações pessoais sensíveis.

1. Coleta de Dados

A extensão pode armazenar informações localmente utilizando o recurso chrome.storage.local do navegador, com a finalidade exclusiva de funcionamento interno da extensão.

Nenhuma informação é enviada para servidores externos.

2. Uso das Informações

Os dados armazenados são utilizados apenas para:

  • sincronização local de fluxos cessuais;
  • backup local de informações;
  • renderização de fluxogramas;
  • exportação de imagens PNG;
  • funcionalidades internas da extensão.

3. Compartilhamento de Dados

A extensão não vende, compartilha ou transfere dados dos usuários para terceiros.

4. Segurança

Todas as informações permanecem armazenadas localmente no navegador do usuário.

5. Permissões do Navegador

Algumas permissões do navegador podem ser utilizadas para permitir a comunicação entre abas e o funcionamento correto da extensão.

6. Alterações nesta Política

Esta Política de Privacidade poderá ser atualizada futuramente para refletir melhorias e novas funcionalidades da extensão.
